# Task Manager CLI

A command-line task manager built with Node.js.

⚠️ This project is under active development.

## Features

- Create tasks
- Edit tasks
- Delete tasks
- Import tasks from CSV
- Export tasks to CSV
- Download tasks from URL

## Technologies

- Node.js
- Inquirer
- Axios
- CSV Parse
- CSV Stringify

## Setup

1. Clone the repository
2. Run:

```bash
npm install
```

3. Create a `.env` file from `.env.example`
